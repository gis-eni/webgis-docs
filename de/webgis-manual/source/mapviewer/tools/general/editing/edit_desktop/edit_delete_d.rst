Bestehendes Objekt löschen
==========================

Um ein bestehendes Objekt zu löschen, muss das Objekt, wie beim Bearbeiten, selektiert werden. 
In der Tabelle, die sich durch die Abfrage des Objektes öffnet, findet man bei den Tabellen-Werkzeugen das Werkzeug *Objekte löschen*.


.. image:: img/loeschen.png

Wählt man dieses Werkzeug, wird in der Vorschau das zu löschende Objekt angezeigt und man hat noch die Möglichkeit, die Lösch-Aktion abzubrechen. 

.. image:: img/loeschen_1.png

.. note::
   Das Löschen wird direkt auf der Geo-Datenbank ausgeführt und ist in der Regel endgültig. Für manche Dienste besteht
   allerdings auch eine Undo Funktion (siehe Abschnitt :ref:`undo_d`).
