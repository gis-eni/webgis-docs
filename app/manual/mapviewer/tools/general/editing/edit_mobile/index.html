Edit im Mobile-Layout¶

Öffnet man das Werkzeug Bearbeiten (Edit) öffnet sich folgender Dialog:

Es gibt unterschiedliche Methoden, ein Objekt zu bearbeiten. Wenn die entsprechenden Geo-Objekte beispielsweise bereits über eine Abfrage ausgewählt wurden, können sie direkt hier bearbeitet werden. Diese ‚Abkürzungen‘ werden später noch beschrieben. Zum Verständnis der Bearbeitungsmöglichkeiten wird hier zuerst auf die drei Standard-Methoden Objekte anlegen, Objekt bearbeiten und Objekte löschen eingegangen. Die einzelnen Vorgänge können über den entsprechenden Button im Werkzeugdialog gestartet werden.

  • Neues Objekt anlegen
  • Bestehendes Objekt bearbeiten
  • Bestehendes Objekt löschen
  • Bearbeitungsschritt rückgängig machen (Undo)
  • Ausgewählte Objekte bearbeiten
    • Geo-Objekte teilen (cut)
    • Multipart Geo-Objekte erstellen (merge)
    • Multipart Geo-Objekt auftrennen (explode)
    • Massenattributierung
