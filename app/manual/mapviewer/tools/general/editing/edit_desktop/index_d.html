Edit im Desktop-Layout

Anhand des Werkzeuges Bearbeiten (Edit) können Geo-Objekte in der Karte erfasst, geändert und gelöscht werden. Dabei können sowohl die Geometrie als auch die Sachdaten des Objektes bearbeitet werden.
Welche Themen und Methoden beim Edit möglich sind, legt der Autor der Karte fest.

Das Bearbeiten von Objekten unterscheidet sich im Desktop-Layout vom Mobile-Layout (siehe Edit im Mobile-Layout). Das Editieren im Desktop-Layout wird in diesem Kapitel genauer beschrieben.
Wählt man in der Toolbox das Werkzeug Bearbeiten (Edit), so öffnet sich folgender Dialog:
In der ersten Zeile befinden sich die Symbole für unterschiedliche Selektion von Geo-Objekten (so können Objekte gleich in diesem Werkzeug selektiert werden und man muss nicht mehr extra in das Abfrage-Werkzeug) und das Symbol für das Anlegen eines neuen Objektes.
Darunter befindet sich ein Drop-Down-Menü, bei dem zwischen den editierbaren Themen der Karte gewechselt werden. Je Thema erscheinen die editierbaren Geo-Objekte in der Liste darunter.

Auf die einzelnen Bearbeitungsmöglichkeiten wird in den folgenden Abschnitten näher eingegangen.
  • Neues Objekt anlegen
  • Bestehendes Objekt bearbeiten
    • Weiter Bearbeitungsmethoden
  • Bestehendes Objekt löschen
  • Bearbeitungsschritt rückgängig machen (Undo)
