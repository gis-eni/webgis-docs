Multipart Geo-Objekte erstellen (merge)¶

Anhand dieses Werkzeuges kann aus mehreren Objekten ein Multipart Objekt erstellt werden, welches ein Geo-Objekt ist, das aus mehreren Abschnitten besteht.
Dazu müssen mindestens zwei Geo-Objekte ausgewählt sein. In der Abfrage-Tabelle dann erscheint das Werkzeug Zusammenführen (merge - selektierte Objekte).

Bemerkung

Nicht für alle Geo-Objekte ist das Erstellen von Multiparts erlaubt! Dies wird vom Karten-Administrator im Vorhinein festgelegt.

Als erstes muss angegeben werden, aus welchem Objekt die Attribute übernommen werden sollen.

Durch Bestätigung mittels des Buttons Zusammenführen wird das bestätigt.

Danach muss man dies noch einmal bestätigen, in dem man auf Zusammengeführtes Objekt: … klickt.

Nun kann man wieder entscheiden, ob man das Ursprungsobjekt beibehalten will oder nicht.
