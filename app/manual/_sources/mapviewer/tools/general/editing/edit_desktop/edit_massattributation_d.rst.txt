Massenattributierung
====================

| Dieses Werkzeug dient dazu, Sachdaten auf mehrere Geo-Objekte gleichzeitig zu übertragen. Dazu müssen diese Objekte über die Abfrage selektiert werden. 
| In der Abfrage-Tabelle erscheint nun das Werkzeug *Massenattributierung (selektierte Objekte)*.

.. image:: img/massenattribut.png

Klickt man dieses an, erscheint der Dialog zu Massenattributierung. Hier ist es wichtig, das ein Häkchen bei den zu ändernden Attributen gesetzt werden muss, so dass das Feld grün hinterlegt ist. Erst dann können Werte eingegeben werden, welche dann auf alle Objekte übernommen werden. 

.. image:: img/massenattribut_1.png

.. note::
    Bei Massenattributierung ist kein *Undo* möglich!
