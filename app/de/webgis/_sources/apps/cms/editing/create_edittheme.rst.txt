Editthema anlegen
=================

Um ein neues Editthema anzulegen, muss im (AGS-)Dienst in den Bereich ``Editing`` gewechselt werden.

.. image:: img/editing1.png

In diesem Bereich sind alle Themen aufgelistet, die in der Kartenanwendung bearbeitet werden können.  
Standardmäßig ist kein Thema innerhalb eines Dienstes bearbeitbar. Erst durch das Hinzufügen eines  
Themas in diesen Bereich/Knoten wird die Bearbeitung durch den Anwender ermöglicht.

.. note::  
   Der CMS-Baum kann an jeder Stelle berechtigt werden. So kann beispielsweise ein bestimmtes Editthema  
   nur einem eingeschränkten Benutzerkreis zur Verfügung stehen. Möchte man das gesamte Bearbeiten schützen,  
   kann die Berechtigung direkt für den Bereich/Knoten ``Editing`` festgelegt werden.

Neues Editthema hinzufügen  
--------------------------

Zum Hinzufügen eines Themas auf den Button ``Neues Edit-Thema`` klicken.

.. image:: img/editing2.png

Hier wird beispielsweise das Thema **„Planungsfläche“** ausgewählt, über das der Anwender eine Fläche  
einzeichnen kann, die das Planungsgebiet eingrenzt.  

Unter *Name/Bezeichnung* kann festgelegt werden, unter welchem Namen das Editthema angezeigt wird.  
Dieser Name muss nicht zwingend mit dem Namen des Darstellungslayers übereinstimmen.  

Zusätzlich zur Geometrie sollten relevante Sachdaten eingegeben werden, z. B.:  
- Planungsnummer  
- Variante  
- Status  
- Verantwortlichkeiten  

Sobald der Anwender eine neue Planungsfläche erstellt, könnte die Bearbeitungsmaske folgendermaßen aussehen:

.. image:: img/editing3.png

### Bearbeitung eines Editthemas  

Nach dem Erstellen erscheint das Editthema in der Liste. Durch Anklicken des Themas stehen folgende  
Bearbeitungsmöglichkeiten zur Verfügung:

.. image:: img/editing4.png

Unter **Edit-Thema/Layer** wird der Verweis auf den Kartendienst-Layer angezeigt, der bearbeitet werden soll.  
Da dieser bereits im vorherigen Schritt beim Erstellen ausgewählt wurde, ist hier keine weitere Aktion erforderlich.
